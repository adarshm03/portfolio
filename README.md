# Adarsh M — Portfolio

> **Build → Break → Optimize.**

Personal portfolio website built with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools — just fast, clean, and deployed.

## 🚀 Live

[adarshm-portfolio.vercel.app](https://adarshm-portfolio.vercel.app)

## ✨ Features

- Scroll-reactive canvas background with animated color-shifting orbs
- Dark / Light mode toggle with system preference detection
- Animated CLI header with typewriter effect
- Hero banner with GIF background and scanline overlay
- Skill icons via Devicons
- Project cards with hover effects
- Education timeline
- GitHub contribution graph (visual)
- Contact form powered by Formspree
- Floating glass dock navigation
- Scroll progress bar
- Fully responsive

## 🗂 Structure

```
public/
├── index.html        # Everything lives here
├── background.gif    # Hero banner
├── profile.jpg       # Avatar
├── bitpad.png        # BitPad project screenshot
└── dsaportal.png     # DSA Portal screenshot
```

## 🛠 Tech

- HTML / CSS / JavaScript — no frameworks
- [Devicons](https://devicon.dev/) — skill icons
- [Formspree](https://formspree.io) — contact form
- [Vercel](https://vercel.com) — deployment
- [Google Fonts](https://fonts.google.com) — Space Grotesk + JetBrains Mono

## 📦 Deploy

```bash
git clone https://github.com/adarshm03/portfolio.git
cd portfolio
# open index.html in browser or use Live Server
```

No install needed. Open `index.html` directly or serve with any static server.
